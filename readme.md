## 🇩🇪 Deutsch

### Die Mission
Loocator ist eine gemeinnützige, Community-getriebene Web-App, die ein alltägliches, aber oft frustrierendes Problem löst: Das schnelle Finden einer zugänglichen Toilette. Egal, ob du dringend ein öffentliches WC, eine rollstuhlgerechte Anlage mit Eurokey-Zugang oder einen Wickeltisch für dein Kind suchst – Loocator zeigt dir den Weg.

### 100% Kostenlos, 100% Frei
Die App wurde aus der Überzeugung heraus entwickelt, dass der Zugang zu sanitären Anlagen ein Grundbedürfnis ist und Informationen darüber frei verfügbar sein müssen.
* **Keine Registrierung:** Du musst kein Konto erstellen.
* **Kein Tracking:** Deine Standortdaten verlassen niemals dein Gerät.
* **Keine Paywalls:** Alle Funktionen, Filter und Routen sind komplett kostenlos.

### Open Data & Community
Loocator greift in Echtzeit auf die weltweiten, freien Daten von **OpenStreetMap (OSM)** zu. Wenn eine Toilette in der Karte fehlt, kannst du sie direkt über die App melden und so aktiv mithelfen, die Datenbank für alle zu verbessern. 

Zusätzlich verfügt Loocator über ein eigenes Rating-System: Nutzer können anonym die Sauberkeit bewerten und rückmelden, ob ein WC tatsächlich zugänglich war. So warnen wir uns gegenseitig vor defekten Anlagen und heben die besten hervor ("Hohe Erfolgsrate").

### Die Technik dahinter
Als **Progressive Web App (PWA)** kann Loocator ohne App-Store direkt auf jedem Smartphone installiert werden und verhält sich wie eine native App. Die Architektur setzt auf Leichtigkeit und Geschwindigkeit: Reines HTML, Vanilla JavaScript und Tailwind CSS sorgen für minimale Ladezeiten – auch bei schlechtem Mobilfunkempfang.

## 🇬🇧 English

### Our Mission
Loocator is a non-profit, community-driven web app designed to solve an everyday yet often frustrating problem: finding an accessible toilet quickly. Whether you urgently need a public restroom, a wheelchair-accessible facility with Eurokey access, or a changing table for your baby – Loocator points you in the right direction.

### 100% Free, 100% Open
This app was built on the belief that access to sanitation is a basic human need, and information about it should be freely available.
* **No Registration:** You don't need to create an account.
* **No Tracking:** Your location data never leaves your device.
* **No Paywalls:** All features, filters, and routing are completely free.

### Open Data & Community
Loocator fetches real-time data from the global, open-source **OpenStreetMap (OSM)** project. If a toilet is missing, you can report it directly through the app, helping to improve the database for everyone. 

Additionally, Loocator features its own rating system: users can anonymously rate cleanliness and confirm if a facility was actually accessible. This helps the community warn each other about out-of-order toilets and highlight the most reliable ones ("High Success Rate").

### The Tech Behind It
Built as a **Progressive Web App (PWA)**, Loocator can be installed directly on any smartphone without needing an app store, feeling just like a native app. The architecture prioritizes lightweight performance: Pure HTML, Vanilla JavaScript, and Tailwind CSS ensure minimal loading times – even on poor mobile networks.
